# exam

To install dependencies:

```bash
bun install
bun add elysia
```

Initialise database:
```bash
bun run src/db/populate.ts
```

To run:
```bash
bun run src/rest/app.ts
```

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
